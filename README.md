# Пупок FM

Офлайн-аудиоплеер для друзей. Загружай MP3/FLAC файлы прямо в браузере – они сохраняются локально (в IndexedDB) и доступны для прослушивания даже без интернета.

## Как пользоваться

1. Открой сайт: `https://твой-логин.github.io/pupok-fm/`
2. Нажми на плюсик ➕ и выбери аудиофайлы (поддерживаются MP3 и FLAC).
3. Управляй воспроизведением, используй поиск.
4. **Чтобы приложение работало офлайн:**
   - Зайди на сайт **хотя бы один раз при включённом интернете**.
   - Дождись полной загрузки.
   - Добавь сайт на главный экран телефона:
     * На Android: в браузере Chrome открой меню → «Добавить на главный экран».
     * На iPhone: в Safari нажми «Поделиться» → «На экран «Домой»».
5. После этого приложение будет открываться и работать **без интернета**!

## Технологии

- Чистый HTML/CSS/JavaScript
- IndexedDB для хранения файлов
- Service Worker для офлайн-доступа
- Адаптивный дизайн, тёмная тема
